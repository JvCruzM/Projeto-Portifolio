# Portfólio Pessoal e Currículo

Projeto desenvolvido como requisito de avaliação acadêmica para o curso de **Sistemas para Internet** na Universidade Católica de Pernambuco (Unicap). Este repositório contém o código-fonte do meu portfólio pessoal, projetado para apresentar minhas habilidades, experiências acadêmicas e projetos em desenvolvimento.

## Visão Geral do Projeto

A aplicação foi construída com foco em performance, componentização e design moderno. A interface utiliza uma identidade visual Dark Mode com realces nas cores roxo, ciano e verde neon, implementando efeitos de *glassmorphism* e transições suaves.

### Principais Funcionalidades

* **Integração com API Externa:** Consumo nativo da [API Pública do GitHub](https://docs.github.com/pt/rest) via Server Components para listar automaticamente os repositórios mais recentes na aba "Projetos".
* **Roteamento Otimizado:** Navegação fluida e sem recarregamento de página utilizando o App Router do Next.js.
* **Design Responsivo:** Layout fluido que se adapta perfeitamente a dispositivos móveis, tablets e desktops.
* **Animações Customizadas:** Utilização do arquivo `template.js` para garantir transições de `fade-in` e `slide-up` em todas as trocas de rota.

---

## Tecnologias e Ferramentas

O desenvolvimento deste projeto utilizou as seguintes tecnologias:

* **[Next.js (App Router)](https://nextjs.org/):** Framework React para renderização do lado do servidor (SSR) e geração de rotas estruturadas.
* **[React](https://react.dev/):** Biblioteca principal para construção da interface de usuário e criação de componentes reutilizáveis (Navbar, Footer, etc).
* **[Tailwind CSS (v4)](https://tailwindcss.com/):** Framework utilitário de CSS para estilização rápida, construção de gradientes complexos e controle de responsividade.
* **JavaScript (ES6+):** Linguagem principal utilizada na lógica de componentes e consumo de dados.
* **Fetch API:** Ferramenta nativa utilizada para requisições HTTP assíncronas.
